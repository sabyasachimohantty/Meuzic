## Backend API Reference

### 🔐 Authentication Endpoints

| Endpoint | Method | Description |
| :--- | :--- | :--- |
| `/auth/signup` | `POST` | Register a new user account. |
| `/auth/login` | `POST` | Authenticate credentials and retrieve tokens. |
| `/auth/refresh` | `POST` | Renew an expired access token using a refresh token. |
